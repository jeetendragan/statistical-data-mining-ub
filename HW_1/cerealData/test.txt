Carbohydrates, sugar and potassium have negative values in some cases. There is 1 cereal with -1 carbo value, 2 cereals with -1 potass value, and 1 cereal with -1 sugar value. Quaker Oatmeal is common in two cases. Two of these cereals belong to the type H. Other than these there is only one other cereal belonging to type H. If we drop the values, then we lose some important information regarding type H cereals. Hence, we set the values to the mean of next closest category, i.e.manufacturer.
